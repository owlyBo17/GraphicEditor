# Графический Редактор (C# WinForms)

Это учебный проект простого графического редактора, разработанного на языке **C#** с использованием технологии **Windows Forms**. Приложение позволяет рисовать на холсте, использовать геометрические фигуры и сохранять результат в виде изображений.

## 🎨 Основные возможности

### Инструменты рисования
* **Карандаш (Pen):** Свободное рисование мышью.
* **Линия (Line):** Создание прямых линий между двумя точками.
* **Прямоугольник (Rect):** Рисование прямоугольных областей.
* **Круг (Circle):** Рисование эллипсов и окружностей.
* **Треугольник (Triangle):** Автоматическое построение равнобедренного треугольника внутри выбранной области.

### Настройки и управление
* **Палитра цветов:** 5 предустановленных цветов (Черный, Красный, Синий, Зеленый, Оранжевый) + кнопка **Custom** для выбора любого цвета из системной палитры Windows.
* **Толщина кисти:** Два режима — **Thin** (2px) и **Thick** (8px).
* **Очистка (Clear):** Мгновенное удаление всех рисунков и возврат к белому холсту.
* **Сохранение (Save):** Экспорт готового рисунка в форматы **PNG** или **JPG**.

## 🛠 Техническое описание

Проект реализован с акцентом на чистоту кода и базовые принципы программирования:

1.  **Класс Form1:** Является основным контейнером приложения, инкапсулирующим логику интерфейса и обработки событий мыши.
2.  **Объект Bitmap:** Используется как внутренний буфер (холст). Это гарантирует, что рисунок не исчезнет при изменении размера окна или его сворачивании.
3.  **Абстракция рисования:** Метод `DrawToBitmap` централизует управление графическим контекстом (`Graphics`), обеспечивая безопасное создание и удаление объектов в памяти.
4.  **Событийная модель:** Логика построена на переопределении стандартных методов `OnMouseDown`, `OnMouseMove` и `OnMouseUp`.
